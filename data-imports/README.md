# data-imports

대시보드에 수동으로 반영하는 원본 데이터를 두는 폴더입니다.

## 경제 캘린더 업데이트

1. 캘린더 엑셀(인베스팅닷컴 등에서 내보낸 파일)을 이 폴더에
   **`calendar.xlsx`** 라는 이름으로 저장합니다. (기존 파일은 덮어쓰면 됩니다)

   엑셀 1행은 헤더여야 하고, 컬럼 순서는 다음과 같습니다:

   | 날짜 | 시간 | 통화 | 국가 | 발표월 | 표시 | 중요도 | 실제값 | 예상값 | Forecast | 이전값 |

   - 날짜: `2026/08/12` 또는 `2026-08-12`
   - 시간: 한국시간(KST) 기준 `HH:MM`
   - 중요도: `3`(=상) / `2`(=중) / `1`(=하)

2. 스크립트 실행:

   ```bash
   python3 scripts/update-calendar.py          # data.js 갱신
   python3 scripts/update-calendar.py --dry     # 미리보기(파일 안 씀)
   ```

3. 결과 확인 후 커밋·푸시(=배포):

   ```bash
   git add data.js && git commit -m "경제 캘린더 업데이트" && git push
   ```

### 스크립트가 자동으로 해주는 것
- 엑셀에 담긴 **날짜 구간만** 교체합니다. 그 밖의 기존 일정은 보존.
- 중요도 3→상 / 2→중 / 1→하 변환, 국가명 정규화(대한민국→한국, 유로 지역→유럽 등).
- **지표 사전과 연결된 발표**(미국 CPI·고용·ISM, 한국 CPI, 중국 무역수지 등)와
  같은 날·같은 국가·같은 개념이면 중복이므로 엑셀 쪽 행을 자동 제외합니다.
- `cal_` 번호를 1부터 다시 매깁니다.

> 새로운 지표를 지표 사전에 추가해 중복 제외 대상을 늘리고 싶으면
> `scripts/update-calendar.py` 의 `CONCEPT` 딕셔너리에 `지표id: (["포함키워드"], ["제외키워드"])` 를 추가하세요.
